작업 가이드 작성 예시​

# 명판 제작 작업 가이드

## 🎯 목표
이 가이드는 명판 제작 작업의 기본 절차를 설명합니다.

## 📋 사전 준비사항

### 필요한 도구
- [ ] AutoCAD
- [ ] 명판 자동화 프로그램
- [ ] Excel 파일

### 파일 준비
1. DXF 템플릿 파일 확인
2. Excel 텍스트 목록 준비
3. 출력 폴더 생성

## ⚠️ 주의사항

:::warning[중요]
작업 전 반드시 파일을 백업하세요.
:::

## 🔧 작업 단계

### 1단계: 프로그램 실행
프로그램을 실행합니다.

### 2단계: 파일 선택
- 템플릿 파일 선택
- Excel 파일 선택

:::tip[팁]
`templates/` 폴더에 미리 준비된 템플릿들이 있습니다.
:::

## 📊 결과 확인

| 항목 | 확인사항 |
|------|----------|
| 파일 생성 | ✅ |
| 텍스트 배치 | ✅ |
| 형식 확인 | ✅ |

## 📞 문제 발생 시

문제가 발생하면 다음을 확인하세요:

1. **파일 형식**: DXF, XLSX 형식인지 확인
2. **권한**: 폴더 쓰기 권한 확인
3. **용량**: 디스크 여유 공간 확인

:::danger[긴급]
시스템 오류 발생 시 즉시 담당자에게 연락하세요.
:::

🎯 작성 팁​

1. 구조화된 작성​

  • 제목을 명확하게 구분
  • 목록과 체크리스트 활용
  • 단계별로 나누어 작성

2. 시각적 강조​

  • 이모지로 섹션 구분
  • 알림 박스로 중요 정보 강조
  • 표로 정보 정리

3. 사용자 친화적​

  • 쉬운 용어 사용
  • 구체적인 예시 제공
  • 단계별 가이드 제공

4. 유지보수 고려​

  • 정기적인 업데이트
  • 피드백 반영
  • 버전 관리
