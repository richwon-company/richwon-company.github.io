CNC로 다양한 소재를 정밀하게 절삭·가공하는 기술입니다.

---

## 적용 가능한 소재

| 소재명   | 약어 |
| -------- | ---- |
| 알루미늄 | AL   |
| 아크릴   | AC   |
| 페놀     | PH   |

---

## 주요 용도/적용 예시

- 명판, 태그, 패널 등 평판 제품 가공
- 설비명판, 기계패널, 각종 산업용 부품 제작
