UV 경화 잉크로 다양한 소재에 직접 인쇄하는 평판 인쇄 기술입니다.

---

## 적용 가능한 소재

| 소재명             | 약어      |
| ------------------ | --------- |
| 알루미늄           | AL        |
| 아크릴             | AC        |
| 페놀               | PH        |
| 폴리카보네이트     | PC        |
| 스테인리스 스틸    | SUS       |
| 시트지             |           |
| 하이박스           | FI-BOX    |

---

## 주요 용도/적용 예시

- 명판, 안내판, 패널 등 평판 인쇄물 제작
- 특수 소재(시트지, 하이박스 등) 인쇄
