# 📝 사내 규정 및 근무 안내

:::info
리치원 컴퍼니는 가족 같은 분위기에서 자율과 책임을 중시합니다.<br/>
아래 안내는 기본 원칙이며, 상황에 따라 사장님과 상의하여 유연하게 조정할 수 있습니다.
:::
